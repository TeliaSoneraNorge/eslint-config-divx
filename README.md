## Installation

```sh
yarn add eslint @telia/eslint-config-divx --dev
```

## Usage

In your `.eslintrc`-file:
```js
{
  "extends": [
    "@telia/eslint-config-divx/frontend"
  ]
}
```

## Usage Backend

In your `.eslintrc`-file:
```js
{
  "extends": [
    "@telia/eslint-config-divx/backend"
  ]
}
```
